delete method

Future delete (dynamic obj)

Удаляет объект

obj может быть:

  • идентификатором объекта ресурса (наследник ObjectId)
  • Map, содержащий параметры запроса

Если методу передается идентификатор объекта, то удаляется указанный объект и возвращается Future с этим объектом.

Если методу передаются параметры запроса, то удаляются все объекты, удовлетворяющие запросу и возвращается Future со списком удаленных объектов.

Implementation 

Future delete(dynamic obj) async {
  String path;
  Map<String, String> queryParameters;

  if (obj is ObjectId) {
    path = '$resourcePath/$obj';
  } else if (obj is Map) {
    path = resourcePath;
    queryParameters = obj;
  }
  final response = await apiClient.delete(
      resourcePath: path, queryParameters: queryParameters);
  return _processResponse(response);
}

replace method

Future<T> replace (T obj)

Полностью обновляет данные

Данные объекта полностью заменяются данными передаваемого объекта. Если какие-то параметры имеют значение null или пустые, то они удаляются

Возвращает Future с измененным объектом

Implementation 

Future<T> replace(T obj) async {
  final response =
      await apiClient.put(resourcePath: resourcePath, body: obj.toJson());
  return _processResponse(response);
}
